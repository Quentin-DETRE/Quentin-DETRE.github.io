Projets

Catcatcatcat

C'est un site web de blog à été réalisé à deux. Les taches étaient réparties de sorte à ce que l'un ait le travail d'un développeur front-end, et l'autre celui du développeur backend. Je me suis occupé pour ma part de la partie backend, et ait donc réalisé une API ainsi qu'une base de donné MySQL rest avec l'aide du framework NestJS. Vous trouverez le code source de l'API en cliquant sur le boutton.

Droidsoft

Droidsoft est un projet d'application mobile android pour le compte du blog en ligne du même nom. C'est une application pour lire les articles qui sont postés sur le site. L'application récupère tout les articles, en permet la lecture, les tries par catégories et possède pleins de fonctionalités, que vous pouvez toutes découvrire en téléchargeant le projet, où en installant l'application depuis le Playstore !

Remindr

Un projet de site web pour apprendre le framework NextJS. Le pricipe de ce site est de pouvoir créer des groupes entre plusieurs utilisateurs et de pouvoir ajouter des Remindr, qui sont des sortes de notes qui envoiet un rappel aux utilisateurs avant que la date spécifié n'arrive. Le rappel est fait par l'envoi de mail 1 semaine avant la date de l'évenement. Work In Progress...
